## 📋Project Overview
This project uses p5.js to create an interactive slot machine.
The slot machine displays random symbols while spinning over time. To control the behavior, it makes use of mouse interactions, animation, and a state system.


## How to run

Use the p5.js editor to run the project or open index.html in a browser.

---

## Controls
Mouse click on the SPIN button → starts the machine
Mouse hover → shows visual effects on the button and the reels
