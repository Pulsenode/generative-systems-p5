## 📋Project Overview
This project creates an animated network using moving particles.
The particles move around the screen and connect to each other with lines when they are close. This creates a dynamic and changing visual effect.


## How to run

Open index.html in a broswer, or run the project using the p5.js editor.

## Test checklist

Confirmed the sketch loads without errors
Confirmed particles move on the screen
Confirmed particles bounce on edges
Confirmed lines are drawn between close particles
Confirmed circles and lines change based on distance
Confirmed mouse click generates a new network
Confirmed colors of particles change
Confirmed the sketch works after refresh

---

## The Process

First, I created a particle system using a class.
Each particle has a position and a velocity, and moves on the screen.
Then, I added a system to connect particles using lines when they are close.
I used distance calculations to control when lines appear.
Random values were used for colors, number of particles, and distance.
Finally, I added a mouse click to generate a new network.
