# A1_M1_Case3_Clement_Dedieu

---

## 📋Project Overview
This project creates a generative loop-based visual using polar coordinates and Perlin noise.
The structure is built using loops to form repeated circular patterns that evolve over time.
An image is displayed in the composition, and words from an in-code array are animated to demonstrate string manipulation.


## How to run

Open index.html in a broswer, or run the project using the p5.js editor.

---

## Controls
Reloading the page or click the play button on p5.js editor

---

## Weekly Skills demonstrated

Loops (to generate shapes)
Animation using draw()
Noise (noise()) for organic movement

---

## Test checklist

Confirmed the sketch loads without errors
Confirmed loops are used to generate repeated shapes
Confirmed animation changes over time
Confirmed noise is used to create variation
Confirmed the sketch works after refresh

---

## The Process

First, I created a loop to draw circular shapes around the center.
Then, I used noise() to change the shape and create a more organic effect.
I added animation using frameCount so the shapes move over time.
Finally, I added a mouse click to refresh the visual.
